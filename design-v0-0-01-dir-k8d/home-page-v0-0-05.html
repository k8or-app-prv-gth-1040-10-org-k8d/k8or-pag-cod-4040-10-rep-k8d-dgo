k8or specializes in containerized operating systems based on Docker images

We take popular pre-built Docker images like MySQL, PostgreSQL, DataStax, MinIO, Nuclio, Portainer, Kong Gateway, Envoy Proxy, n8n, Dapr, Kafka, MongoDB, Redis, Zipkin, and others, and perform a process called "k8orization".

Pure OS K8orization

Creation of k8or-specific users and implementation of enhanced security measures.
Test an OS Image
Package File K8orization

Elimination of superfluous elements from pre-installed packages during base image creation.
Test a Package Image
Application File K8orization

Trimming down the application files of specific tools by removing extraneous components.
Test an Application Image
Microservice File K8orization

Streamlining microservice files within Docker images by purging redundant elements, bolstering security, establishing service profiles, and restricting unnecessary system calls.
Test a Microservice Image
